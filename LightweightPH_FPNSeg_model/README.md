# Proposed Lightweight PH_FPNSeg model

### Prerequisites
The model requires Python 3.9 or later and a CUDA-enabled GPU

### Running the MIC-Based Proposed Model
To run the Lightweight PH_FPNSeg model, use the slum script, or just run:
- For training:
  ```bash
  python train.py
  ```
- For testing:
  ```bash
  python test.py
  ```
